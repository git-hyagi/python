# some basic command to use git
* show project status (cached/modified/new/etc)
~~~
git status
~~~

* show diff between commits
~~~
git diff
~~~

* add files to be commited
~~~
git add <file>
~~~

* commit the modifications
~~~
git commit -m "<Commit Message>"
~~~

* push to the remote repo
~~~
git push
~~~

~~~
Commit message must be in imperative form and represents what functionality this commit will implement:
"<Commit Message>"
~~~
